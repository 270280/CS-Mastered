CS Mastered

Not just learned but Mastered

Day 1
Intro to HTML/CSS/JavaScript + Hello World program

Students learn about the basics of coding and get everyting set up for coding. They then build a Hello World program using basic coding.
Day 2
Basics of HTML

Students learn about the basics of HTML, including headings, tables, semantics, and images.
Day 3
Basics of CSS

Students learn about CSS, which gives a website its style. They learn about the different kinds of selectors and modifers, as well as padding, margin, width, and height.
Day 4
Mini-project 1: About Me webpage

Students create an About Me site using the HTML and CSS that they learned so far.
Day 5
Into to JavaScript

Students learn about basic Javascript, like variables, if statements, and for loops.
Day 6
Mini-project 2: Make a Quiz

Students learn how to make a quiz, and have the computer evaluate if the answer is right or wrong.

Day 7
Intro to HTML canvas

Students learn how to draw shapes on a webpage using HTML canvas.
Day 8
A Deeper Dive into JavaScript

Students learn more about Javascript and OOP coding. They learn constuctors, class, objects, and event listners
Day 9
Final Project: Catch Game: Day 1

Students begin working on a simple catch game in javascript, where there are circles falling from the top and you have to move a square to catch them.
Day 10
Final Project: Catch Game: Day 2 / What to do next

Students continue working on a simple catch game in javascript. They also learn about the next steps they could take to become an even better coder.
